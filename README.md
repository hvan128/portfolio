# Portfolio Website - Ngô Hải Văn

Trang web cá nhân được xây dựng với Next.js 14 và Tailwind CSS.

## Cài đặt

```bash
npm install
```

## Chạy dự án

```bash
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000) trong trình duyệt để xem kết quả.

## Build cho production

```bash
npm run build
npm start
```

## Deploy lên VPS

Xem hướng dẫn chi tiết tại [DEPLOY.md](./DEPLOY.md) để deploy project lên VPS với CI/CD tự động.
